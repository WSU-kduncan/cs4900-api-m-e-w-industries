# API Routes for Gamer Match

## GET Methods

### User Entity

- All Users
- Users by Id
- Users by First Name

### Game Entity

- All games
- Games by Id
- Games by Title

### Matched_User Entity

- All Matched Users (For a user)

## POST Methods

### User Entity

- Create New User

## PUT Methods

### User Entity

- Update User
- Update Matched_User (Is_Matched field)

## DELETE Methods

### User Entity

- Delete User
